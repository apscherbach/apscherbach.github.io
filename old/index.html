Proxmox VE 8.x

If you're interested in learning about virtualization, you've come to the right place. If you'd also like to learn about Proxmox, you're in the right place. The course is designed to develop activities sequentially, where actions performed in one class have an impact on subsequent classes. This is how the course context is set up and presented.

Guide of OPNSense Firewall

I'd like to welcome you in advance to our Basic Guide to the OPNSENSE Firewall, where we'll explore this tool and develop skills for your professional or personal development. Believe me, starting with the basics will put you ahead of the curve.
